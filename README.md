## Requirements
- [python](https://www.python.org/downloads/release/python-3140/) 
- tkinter
```bash
pip install tkinter
```

```bash
pip install rich
```

## Panduan Instalasi

```bash
git clone https://github.com/rizZkaa-png/Tugas-Version-Control-System.git
```

```bash
 cd Tugas-Version-Control-System
```
